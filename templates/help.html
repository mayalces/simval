Introduction

About GSGMV

GSGMV is a web tool that allow get genetic data in text files with the specific format for MapDisto, OneMap and MSTMap. Also, it allow to validate the genetic maps based on the genetic data simulation.

Genetic Data Simulator

How to simulate genetic data?

To simulate genetic data, the user must choose the Simulator tab and must fill the following fields in the form:

  • Chromosome quantity
  • Chromosome size
  • Population size
  • Markers quantity
  • Ploidy
  • Recombination rate
  • Mutation rate
  • Error rate
  • Seed

Once the fields are filled, the user must choose the file format, depending on the tool: MapDisto, OneMap, MSTMap. Finally, the user must click on the Simulate button.

Genetic Map Validator

How to validate genetic maps?

To validate genetic maps, the user must choose the Validator tab and must fill the following fields in the form:

  • Upload Genetic Map File
  • Upload Simulated Genetic Map File
  • Choose the comparison criteria
    • Edit Distance
    • Kendall Tau
    • Graphic Comparison

Once the fields are filled, the user must click on the Validate button.
